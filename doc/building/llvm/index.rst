LLVM Backend
############

.. rubric:: Requirements

* GCC (Gnu Compiler Collection)
* GNAT (Ada compiler for GCC)
* LLVM (Low-Level-Virtual Machine) and CLANG (Compiler front-end for LLVM)

  * 3.5
  * 3.8
  * 3.9
  * 4.0
  * 5.0

.. rubric:: Supported platforms

* :doc:`GNULinux-GNAT`
* :doc:`Windows-MinGW-GNAT`
* Mac OS
